# PROJET FINAL ELLIOT GAULIN

## Installation du projet
1. Installation de la base de données  
    ```mongorestore --uri="mongodb://localhost:27017/ma-bd" --dir="./dump/test"```
1. Installation d'express
    ```npm install```
1. Lancement du serveur
    ```npm start```
